# 飞书新闻机器人

每日自动向飞书发送"今日排行前十新闻"的 Python 机器人，支持 GitHub Actions 云端自动运行。

## 功能特性

- 📰 自动抓取百度热搜新闻（主备容灾）
- 🔍 智能比对昨日新闻，避免重复推送
- 📤 部分更新时只发送新新闻
- 🛡️ 异常拦截机制，保护历史记录
- 💬 精美飞书卡片排版（带摘要）
- ⏰ GitHub Actions 每日定时运行（北京时间 9:00）

## 项目结构

```
feishu-news-bot/
├── feishu_news_bot.py      # 核心业务逻辑
├── requirements.txt        # Python 依赖
├── README.md              # 项目说明
├── .github/
│   └── workflows/
│       └── run_bot.yml    # GitHub Actions 配置
└── news_history.json      # 历史记录（自动生成）
```

## 数据源

### 主渠道（优先）
- 接口：`https://api.vvhan.com/api/hotlist/baidu`
- 特点：免费、无需API Key、返回结构化JSON

### 备用渠道（容灾）
- 目标：`https://top.baidu.com/board?tab=realtime`
- 方式：BeautifulSoup 网页爬虫
- 特点：纯爬虫方案，零依赖

## 本地测试

```bash
# 安装依赖
pip install -r requirements.txt

# 设置环境变量
export FEISHU_WEBHOOK='https://open.feishu.cn/open-apis/bot/v2/hook/你的webhook地址'

# 运行脚本
python feishu_news_bot.py
```

## GitHub Actions 配置

1. 在 GitHub 仓库的 Settings -> Secrets and variables -> Actions 中添加 Secret：
   - 名称：`FEISHU_WEBHOOK`
   - 值：飞书机器人的 Webhook 地址

2. 推送代码后，机器人会每天北京时间 9:00 自动运行

## 飞书 Webhook 获取方式

1. 打开飞书，进入目标群组
2. 点击群组设置 -> 群机器人 -> 添加机器人
3. 选择"自定义机器人"
4. 复制 Webhook 地址

## 发送逻辑

- **完全重复**：今天10条新闻与昨天完全相同 → 发送"今天与昨日新闻一样"
- **部分更新**：有新新闻出现 → 只发送新新闻列表（带摘要）
- **数据源异常**：所有渠道获取失败 → 发送红色告警卡片，跳过历史记录更新
- **状态更新**：每次运行后都会更新 `news_history.json`

## 飞书卡片样式

### 新闻卡片（蓝色主题）
```
🚀 今日新闻速递
───────────────────────────────
📰 今日排行前十新闻（更新部分）
───────────────────────────────
1. 新闻标题
   这里是新闻摘要内容（灰色显示）
   
2. 新闻标题
   这里是新闻摘要内容（灰色显示）
───────────────────────────────
更新时间：2024-01-01 09:00:00
```

### 重复卡片（灰色主题）
```
📋 今日新闻速递
───────────────────────────────
今天与昨日新闻一样，没有新内容更新。
───────────────────────────────
检查时间：2024-01-01 09:00:00
```

### 告警卡片（红色主题）
```
⚠️ 系统告警
───────────────────────────────
API获取异常，今日暂不更新历史记录
───────────────────────────────
告警时间：2024-01-01 09:00:00
```
